# Chat View 빠른 시작 가이드

## 1. 사전 요구사항

- Node.js 18+ 설치
- chat-platform 백엔드 서버 실행 중

## 2. 설치

```bash
cd chat-view
npm install
```

## 3. 환경 변수 설정

`.env` 파일 생성:

```env
NUXT_PUBLIC_API_BASE=http://localhost:8080
NUXT_PUBLIC_WS_BASE=ws://localhost:8082
```

## 4. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 http://localhost:3000 열기

## 5. 첫 사용

### 5-1. 로그인
1. http://localhost:3000/login 접속
2. 백엔드에 등록된 계정으로 로그인
3. 로그인 성공 시 메인 페이지로 이동

### 5-2. 채널 확인
- 왼쪽 사이드바에서 내 채널 목록 확인
- 채널 클릭으로 입장

### 5-3. 메시지 보내기
- 하단 입력창에 메시지 입력
- Enter로 전송 (Shift+Enter로 줄바꿈)

### 5-4. 실시간 메시지
- 채널 입장 시 자동으로 WebSocket 연결
- 다른 사용자의 메시지가 실시간으로 표시됨

## 8. 문제 해결

### 로그인 실패
- 백엔드 서버 실행 확인 (포트 8080)
- 계정 정보 확인
- 브라우저 콘솔에서 에러 확인

### 채널 목록이 안 보임
- 네트워크 탭에서 API 호출 확인
- 토큰 만료 확인
- 로그아웃 후 재로그인

### WebSocket 연결 실패
- WebSocket 서버 실행 확인 (포트 8082)
- 브라우저 콘솔에서 에러 확인
